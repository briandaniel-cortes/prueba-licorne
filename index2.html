Prueba técnica

Bienvenido!

En esta prueba técnica evaluaremos tu nivel y conocimiento en el mundo del desarrollo web.

Esta prueba consta de 3 fases en las que aplicarás tus conocimientos en los siguientes temas

Programación Front End

Programación Back End

Diseño Web

Instrucciones

  1. Crear una rama nueva en el repositorio de Github con el siguiente formato Nombre_Apellido
  2. Deberás diseñar la vista para un carrito de compras que contenga el ID, imagen, precio y nombre del producto, no se puede utilizar librerías de css o frameworks de js, el diseño y la maquetación deberán relizarse únicamente con HTML, CSS / SASS / SCSS y JavaScript
  3. Al terminar el diseño web del carrito de compras deberás consumir el siguiente servicio web https://licorne.mx/prueba/api.php, cumpliendo con lo siguiente:
    1. El consumo se deberá realizar por medio de JavaScript
    2. Deberás utilizar el método GET
    3. Con la data obtenida del servicio web deberás llenar tu carrito de compras
  4. Al finalizar deberás crear un programa en PHP o Python que obtenga la suma de cada número en la siguiente lista [1, 2, 3, 4, 5] imprimiendo cada valor junto con su operación en pantalla
      Ejemplo:
    • 1 + 1 = 2
    • 1 + 2 = 3
  5. Por último deberás subir tu prueba a la rama de Github que creaste en el paso 1
